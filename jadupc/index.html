What is JaduPc?

JaduPc is a Bangladeshi startup initiative producing low cost computers with ARM based SBCs (single board computers) - running a custom operating system (Shopno OS) based on Linux that resembles Windows. The low cost, dependable JaduPc is tailored for youth education, providing access digital literacy, programming and even robotics. JaduPc aims to arm Bangladeshi youth with the tools for skill building.

Technical Specifications

  • Powerful Processor

    Quad-core ARM Cortex A53 supported upto 1.8 GHz

  • Operating System

    Custom operating system with preinstalled educational software for learning

  • Ports

    3 USB , 1 HDMI Port, 1 Micro SD Slot and a Lan Port

  • Memory and Storage

    4GB DDR3 Ram with 32/64 GB Rom expandable upto 1 TB

  • Display

    19 inch TFT LED Monitor with 1366x786 resolution

  • Connectivity

    Wifi 2.4-5G
